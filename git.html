Commandes Git

Versionner son travail

Versionner en Local

Commandes Description
git init Initialise le dépôt
(se mettre sur le bon dossier),
mieux à faire depuis github.com
git add . Ajoute toutes les
modifications le "." symbolise TOUT
git commit -m "explication Créer un nouveau commit.
git add Pousse les fichier en zone d'index,
git commmit les sauvegarde réellement dans un
nouveau commmit

Gérer les commits

Commandes Description
git log Liste des commmits
git log -n2 Affiche les deux derniers commits
git show Voir un commit spécifique
git checkout "sha-1" Remettre la version du "sha-1"
git checkout main Remmettre la version la plus récente

Versionner sur un dépôt distant

Commandes Description
git clone (lien-github.com) Récupérer le travail depuis un dépôt distant
git push -u origin main Pousse les modifications vers le serveur
git push -f origin main Pousse de force des modifications
(à manipuler avec précaution

Naviguer dans Git Bash

Commande Description
pwd Savoir dans quel dossier je suis
mkdir "dossier" Créer un dossier (Make Directory)
touch "fichier.txt Créer un Fichier
ls Liste le dossier courant
ls -la liste tout plus précisément que ls
cd "dossier" Allez dans le dossier (change Directory)
cd .. Remonter d'un dossier
